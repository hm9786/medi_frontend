# 한글 폰트 파일

이 폴더에 다음 폰트 파일을 추가하세요:

## Noto Sans KR (권장)

1. https://fonts.google.com/noto/specimen/Noto+Sans+KR 에서 폰트 다운로드
2. 또는 다음 링크에서 직접 다운로드:
   - Regular: https://fonts.gstatic.com/s/notosanskr/v36/PbykFmXiEBPT4ITbgNA5Cgm20HTs4JMMuA.woff2
   - Bold: https://fonts.gstatic.com/s/notosanskr/v36/PbykFmXiEBPT4ITbgNA5Cgm20HTs4JMMuA.woff2

3. 다운로드한 파일을 다음 이름으로 저장:
   - `NotoSansKR-Regular.ttf` (또는 `.woff2` 파일을 `.ttf`로 변환)
   - `NotoSansKR-Bold.ttf`

## 나눔고딕 (대안)

1. https://hangeul.naver.com/2017/nanum 에서 폰트 다운로드
2. 다운로드한 파일을 다음 이름으로 저장:
   - `NanumGothic-Regular.ttf`
   - `NanumGothic-Bold.ttf`

## 참고

- `.woff2` 파일은 `.ttf`로 변환해야 할 수 있습니다.
- 온라인 변환 도구: https://convertio.co/woff2-ttf/
